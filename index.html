p5.js Showcase

Purpose: How might we showcase a project to increase access to the open source p5.js library and inspire others to contribute to Processing's creative coding community?

Solution: p5js.org/showcase — A Google Summer of Code 2019 project to create a new 6-part section on the p5.js documentation website that showcases creative and inclusive uses of p5.js and encourages project nominations; officially released in February 2020 with the first p5.js major release version and curated in 2020-2022 by student contributors Connie Liu, Katie Chan, and Annie Zheng

Contribution:

  • Researched existing examples of community-based project showcases, organized research on how p5.js is used in the wild into categories of p5.js contributions, and established a straightforward form-based process for creators to nominate a project for the Showcase
  • Designed wireframes, created a prototype, and implemented using HTML, CSS, and Node.js templating to support internationalization
  • Curated 6 projects (out of 60+) and interviewed their creators (out of 5+ million users!) about their groundbreaking process of using p5.js to explore art and design concepts and mediums, p5.js libraries and other technologies, or ways of learning and teaching: Roni Cantor, Yona Ngo, Dae In Chung, Casey Conchinha + Louise Less√©l, Moon Jang + Xin Xin's UGA students, Qianqian Ye
  • Advocated for an accessibility review, leading to a site-wide audit enagagement with accessibility subject matter experts Sina Bahram and Claire Kearney-Volpe and a code refactor with mentor Kate Hollenbach and p5.js creator Lauren Lee McCarthy
  • Featured in p5.js version 1.0, Processing's blog, Processing's open source software contributor documentation, p5.js 1.0 Contributors Zine, p5.js 1.0 Release Party 2020 @ USC, Processing Community Day 2021 @ Seoul, and 20th Anniversary Processing Community Catalog
