p5.js Showcase

Purpose: How might we showcase a p5.js-based project to inspire others to use the open source JavaScript library and contribute to the Processing Foundation community? How might we make p5.js more accessible?

Solution: p5js.org/showcase — A Google Summer of Code 2019 project to create a new section on the p5.js documentation website that showcases creative and inclusive uses of p5.js and encourages submissions; officially released in February 2020 with p5.js version 1.0 and updated in summer 2020

Contribution:

  • Researched examples of community showcases and p5.js contributions, designed wireframes and prototypes, and implemented using HTML, CSS, and templating and internationalization with Node.js, Grunt, YAML, and Assemble
  • Advocated for an accessibility review during a site-wide audit by accessibility specialists and refactored design and code
  • Published project documentation and write-up on Processing’s blog, featured in p5.js 1.0 Contributors Zine, and invited to speak at the p5.js 1.0 Release Party
